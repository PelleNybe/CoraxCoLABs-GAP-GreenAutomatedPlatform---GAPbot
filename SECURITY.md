# Security Policy

## Supported Versions

The following versions of the Corax CoLAB GAP system are currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of our architecture and software very seriously, especially concerning our decentralized autonomous swarms and Zero-Trust infrastructure. If you have discovered a security vulnerability in this repository, the GAP system, the B.A.T.M.A.N.-adv mesh network configurations, or the Mission Control dashboard, please **do not** report it by creating a public GitHub issue.

Instead, please report it via one of the following methods:

1. **Email:** Reach out securely to the lead developer Pelle Nyberg through the contact information provided on [Corax CoLAB](https://coraxcolab.com).
2. **Direct Message:** Contact Pelle Nyberg directly via [LinkedIn](https://www.linkedin.com/in/pellenyberg/).

Please include the following information in your report:
* The type of vulnerability (e.g., Cryptographic flaw in Zero-Trust handshake, RCE, network spoofing).
* The steps required to reproduce the vulnerability.
* Any applicable code snippets or proof-of-concept (PoC).
* The potential impact if the vulnerability is exploited on an autonomous physical system.

We will endeavor to respond to your report within 48 hours and provide an estimated timeline for remediation. We appreciate your efforts to responsibly disclose vulnerabilities and keep the community safe.
